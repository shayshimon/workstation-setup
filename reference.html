🛠️ Workstation Reference

Shell & Prompt

⭐ Starship

prompt

Minimal prompt — shows git, python venv, node, aws profile only when relevant. Zero-config out of the box.

Config: ~/.config/starship.toml

📂 direnv

env vars

Auto-loads/unloads environment variables when you enter/leave a directory.

echo 'export AWS_PROFILE=prod' > .envrc
direnv allow
# now cd in/out auto-sets vars

🚀 zoxide

smart cd

Remembers directories you visit. Fuzzy-jump anywhere.

z projects       # jump to best match
z atomic ppc     # multi-term fuzzy
zi               # interactive picker

Search & Navigation

🔍 fzf

fuzzy finder

Fuzzy search everything — history, files, directories.

Ctrl+R    fuzzy command history
Ctrl+T    fuzzy find files
Alt+C     fuzzy cd into subdirs
vim $(fzf)  open searched file

⚡ ripgrep (rg)

search text

Blazingly fast recursive text search. Respects .gitignore.

rg "def train"           # recursive search
rg "TODO" --type py      # only Python files
rg "error" -i -l         # case-insensitive, filenames only
rg "pattern" -C 3        # 3 lines context

📁 fd

find files

Fast, user-friendly alternative to find.

fd "*.py"                # find Python files
fd migration --type d    # find directories
fd --extension json      # by extension
fd -H ".env"             # include hidden

File Viewing

🦇 bat

cat + syntax

cat with syntax highlighting, line numbers, git changes.

bat file.py            # view with colors
bat -l json data.txt   # force language
bat --diff file.py     # show git changes
bat -p file.py         # plain output

📋 eza

modern ls

ls replacement with colors, icons, git status.

eza                  # simple ls
eza -la              # long + hidden
eza --tree --level=2 # tree view
eza -l --git         # git status column

🔧 jq

JSON

Command-line JSON processor — filter, transform, pretty-print.

cat data.json | jq '.'             # pretty-print
cat data.json | jq '.items[0].name' # extract
curl -s api/url | jq '.[] | .id'   # pipe API
jq -r '.key' file.json             # raw (no quotes)

📖 tldr

man pages

Community-driven simplified man pages with practical examples.

tldr tar          # common tar recipes
tldr git rebase   # concise examples

Git

🐙 gh

GitHub CLI

Full GitHub from terminal — PRs, issues, CI, repos.

gh pr create          # create PR
gh pr list            # open PRs
gh pr checkout 42     # checkout PR locally
gh issue list         # issues
gh repo view --web    # open in browser
gh run list           # CI status

🌳 lazygit

terminal UI

Full git GUI in the terminal. Just run lazygit.

Space     stage/unstage
c         commit
p         push
P         pull
?         help
q         quit

🎨 git-delta

better diffs

Syntax-highlighted, side-by-side diffs. The setup script configures this when CONFIGURE_GIT=true.

Equivalent ~/.gitconfig settings:

[core]
    pager = delta
[interactive]
    diffFilter = delta --color-only
[delta]
    navigate = true
    side-by-side = true

HTTP

🌐 httpie

API client

Human-friendly HTTP client. Cleaner than curl for APIs.

http GET api.com/users             # GET
http POST api.com/users name=Shay  # POST JSON
http -a user:pass GET api.com/data # auth
http --headers GET example.com     # headers only

Python

🐍 uv

Python toolkit

Manage Python versions, virtual environments, and Python CLI tools.

uv python install 3.12      # install Python
uv venv                     # create .venv
source .venv/bin/activate   # activate
uv pip install requests     # install into venv
uv tool list                # see installed tools
uv tool upgrade --all       # update tools

Node & AWS

📗 nvm

Node versions 
nvm install --lts       # latest LTS
nvm use 20              # switch
nvm alias default 20    # set default

🔐 aws-vault

credentials

Securely store AWS creds. Handles MFA/assume-role.

aws-vault add prod                    # store creds
aws-vault exec prod -- aws s3 ls      # run with creds
aws-vault login prod                  # open console

Database

🗄️ mycli

MariaDB/MySQL

MySQL client with autocomplete, syntax highlighting, pager.

mycli -h host -u user -p -D dbname
# Tab      autocomplete tables/columns
# Ctrl+D   exit
# \G       vertical output

work() Command

🚀 work

dev layout

Creates a 4-pane tmux layout for any repo.

cd ~/projects/my-repo
work
┌──────────┬─────────────────┐
│  codex   │                 │
├──────────┤      nvim       │
│  claude  │                 │
├──────────┤                 │
│ terminal │                 │
└──────────┴─────────────────┘

Re-running work in same dir just switches to existing window.
